## 1. エグゼクティブサマリー

LinguaSync は、日本国内の現場で働く外国人スタッフと日本人管理者の間の **言語の壁** を、リアルタイム翻訳と「やさしい日本語」変換で解消するクラウド SaaS です。

**ご提案の概要：**

- **MVP 開発費 ¥5,000,000 (一括) + 月額 ¥12,800/店舗 (Business プラン)** からご利用いただけます
- 即日導入可能。サーバ構築・保守作業は一切不要
- 日本国内 (東京) のデータセンターで稼働
- 6 言語自動翻訳 (日本語→英語 / ベトナム語 / インドネシア語 / ミャンマー語 / ネパール語)
- 個人情報保護法 / GDPR 準拠の運用体制

同等のシステムを SI 開発で新規発注する場合、約 **¥45,000,000〜¥200,000,000 の開発費 + 6〜12 ヶ月の開発期間** が必要となります。LinguaSync は SaaS モデルにより、これを **約 1/4〜1/15 のコスト** で即日ご利用いただけます。

## 2. 課題と提供価値

### 2.1 解決する課題

| 現場の課題 | 業務への影響 |
| --- | --- |
| 外国人スタッフへの指示が伝わらない | 業務ミス・配膳ミス・接客クレーム |
| 言語の違いによる安全事故リスク | 厨房 / 食品衛生 / 危険物の事故 |
| マネージャーが英語等の言語を話せない | 採用後の早期離職、教育コスト増 |
| 多店舗で標準化された業務指示が困難 | 店舗ごとのサービス品質バラつき |
| 既読・未読・理解度の確認ができない | 重要指示の伝達失敗 |

### 2.2 LinguaSync が提供する価値

1. **日本人管理者は日本語のまま入力**　→ 自動的に各スタッフの母国語に翻訳して配信
2. **3 ボタン応答 (わかりました / もう一度 / わかりません)**　→ 理解度を即時把握
3. **店舗専用の業務用語辞書**　→ 「3番テーブル」「サイドメニュー」等の固有表現も正確に翻訳
4. **やさしい日本語自動変換**　→ 日本語学習中のスタッフにも理解しやすく
5. **応答履歴の蓄積**　→ 「未応答」のスタッフを自動リスト化、教育機会の発見

## 3. サービス概要

### 3.1 提供形態

- 完全クラウド SaaS (PWA / iOS・Android 対応)
- 専用アプリのインストール不要 (ブラウザでアクセス)
- 管理者・現場スタッフでそれぞれ専用画面

### 3.2 標準機能 (全プラン共通)

| カテゴリ | 機能 |
| --- | --- |
| **コミュニケーション** | リアルタイム指示送受信 (WebSocket) ／ 6 言語自動翻訳 ／ やさしい日本語変換 ／ 3 ボタン応答 ／ カスタム音声応答 ／ 画像添付指示 |
| **スタッフ管理** | 個人スタッフアカウント ／ グループ管理 ／ CSV 一括登録 ／ アバター画像 |
| **履歴・分析** | 60 日分の指示・応答履歴 ／ 応答内訳の自動集計 ／ 「未応答」スタッフ一覧 |
| **管理者画面** | PC / モバイル対応 ／ 6 言語UI切替 ／ 受信者選択 (全員 / グループ / 個別) |
| **データ管理** | データエクスポート (JSON) ／ 削除依頼対応 (5 営業日以内) |

### 3.3 対応言語

- 日本語 (原文)
- 英語
- ベトナム語
- インドネシア語
- ミャンマー語
- ネパール語

※ タイ語・タガログ語等の追加は、オプションで対応可能 (第 9 節参照)。

## 10. セキュリティ・運用体制

### 10.1 暗号化

- **通信経路：** TLS 1.2 以上の強制 (HTTPS / WSS)
- **データ at-rest：** AES-256 (AWS KMS による暗号化)
- **パスワード：** bcrypt ハッシュ化
- **セッショントークン：** JWT (HS256)、ステートレス設計

### 10.2 データ分離

- 全データはワークスペース ID (UUID) でスコープ管理
- アプリケーション層およびデータベース層 (Enterprise: Postgres RLS) の二重分離
- 他社データへの到達経路なし

### 10.3 削除依頼対応

専用 API により、以下を不可逆に削除します。
- ワークスペース情報
- 全スタッフアカウント (個人情報・パスワードハッシュ)
- 指示・応答履歴・プレゼンスログ
- 専用用語集・チャット履歴
- アバター画像・指示画像

削除前のデータエクスポート (JSON 形式) も同 API で提供可能です。

### 10.4 DPA・セキュリティチェックシート対応

- 個人情報保護委員会の点検シート対応可能
- PMS / ISMS 体系のチェック項目対応可能
- 貴社指定書式 (NDA · 業務委託契約 · 個別 DPA) への回答可能 (Business プラン以上)
- 詳細は別添「SECURITY.md (セキュリティおよび運用ポリシー)」をご参照ください。

### 10.5 サービス継続性

- 99.5%〜99.9% 稼働率目標 (プランによる)
- 日次バックアップ (Supabase 標準)
