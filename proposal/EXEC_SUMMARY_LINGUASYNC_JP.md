# LinguaSync — 多言語スタッフ・コミュニケーション SaaS

> **店舗の「言葉の壁」を、5 分で消す。**
> 6 言語リアルタイム翻訳 × 個人ログイン × 既読・応答管理。日本リージョン保管。

---

## 経営サマリー (1 ページ版)

### 課題 — なぜ今、解決すべきか

| 業界共通の現場課題 | LinguaSync が解く |
|---|---|
| 外国人スタッフが増え、日本語指示が伝わらない | 6 言語 (日 / 英 / 韓 / 中 / 越 / インドネシア) **自動翻訳** |
| 「了解」「分からない」が紙・口頭で漏れる | **応答ボタン** + 既読管理で漏れゼロ |
| QR コード共有方式は誰が誰だか不明 | **個人ログイン** で本人確認・履歴追跡 |
| 店舗ごとに情報が分散・本部の可視化が困難 | **ワークスペース分離** + 本部スーパー管理 |

---

### サービス — 何を提供するか

**3 階層 UI** で店舗・本部・運営を分離

| ロール | できること |
|---|---|
| 👤 スタッフ (現場) | 自分の言語でログイン → 指示を母国語で受信 → 1 タップで応答 |
| 👨‍💼 店舗マネージャー | スタッフ追加 (CSV 一括) / 指示送信 / リアルタイム応答ボード |
| 🏢 本部スーパー管理者 | 全店一覧 / 月次請求レポート (店舗別 利益率) / API 使用量モニタ |

**主要機能** (MVP 標準搭載)

✅ 6 言語リアルタイム自動翻訳 (Google Cloud Translation)
✅ 翻訳キャッシュで API コスト削減 (2 回目以降 0 円)
✅ 画像付き指示 + 応答 (了解 / 質問 / NG)
✅ 個人ログイン (再ログイン後も応答状態維持)
✅ 日次自動バックアップ + JWT セッション
✅ **データは日本国内 (Supabase 東京リージョン) で保管**
