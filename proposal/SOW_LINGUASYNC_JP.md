## 2. プロジェクト概要

### 2.1 目的
[貴社] が運営する店舗における多言語スタッフ間のリアルタイムコミュニケーション課題を解決するため、SaaS 型サービス **「LinguaSync」** の開発および導入を行う。

### 2.2 背景
- 貴社運営店舗において外国人スタッフが [N] 名以上在籍
- 既存のチャットツールでは多言語対応・指示の確認状況管理が困難
- 紙・口頭ベースの引き継ぎでは情報伝達漏れ・誤訳が発生

### 2.3 ゴール
- 6 言語 (日本語 / 英語 / 韓国語 / 中国語 / ベトナム語 / インドネシア語) のリアルタイム自動翻訳
- 管理者からスタッフへの指示と「了解 / 質問 / NG」応答管理
- スタッフごとの個人ログインアカウント (再ログイン後も応答状態を引き継ぐ)
- 店舗単位 (ワークスペース) でのデータ分離・本部側の請求一元管理

---

## 3. 作業範囲 (スコープ)

### 3.1 含まれる作業 (In Scope)

#### Phase 1: MVP 開発 (一括 ¥5,000,000)
**管理者ページ**
- スタッフアカウント管理 (個別追加 / CSV 一括アップロード)
- 指示送信 (テキスト + 画像 + URL)
- リアルタイム応答ボード (了解 / 質問 / NG)
- 応答履歴・既読管理 (再ログイン後も維持)
- 店舗グループ・タグ管理

**スタッフページ**
- 個人ログイン (任意言語選択 / 6 言語)
- 指示の自動翻訳表示 (キャッシュにより 2 回目以降は瞬時)
- 応答ボタン (回答後はロック+バッジ表示)
- 質問・コメント返信
- アイコン画像アップロード

**スーパー管理者ページ (貴社本部向け)**
- 全店舗ワークスペース一覧
- 請求レポート (店舗別 / 月次 / 利益率付き)
- API 使用量モニタリング (翻訳文字数の店舗別追跡)
- データエクスポート (CSV / JSON)
- 店舗単位の課金プラン設定

**インフラ・運用**
- カスタムドメイン設定 (1 ドメイン)
- 日本リージョンデータベース (Supabase Tokyo)
- HTTPS (SSL 証明書) 自動更新
- 自動バックアップ (日次)
- 6 言語 PWA 対応 (スマホ・タブレットで動作)

#### Phase 2: 運用・保守 (月額 SaaS 料金)
- 24/7 サービス稼働 (SLA 99.5%)
- 自動バックアップ + 障害時復旧
- セキュリティパッチ適用
- 軽微な不具合修正 (現行仕様の範囲)
- メールサポート (営業日 2 時間以内一次返信)
- 月次稼働レポート + 月次請求書

### 3.2 含まれない作業 (Out of Scope)

以下は本 SOW に含まれず、別途見積:
- 既存システム (POS / 勤怠 / シフト / 給与) との連携開発
- 店舗別カスタム UI / カスタム機能
- 専用モバイルアプリ (iOS / Android ネイティブ)
- 7 言語目以降の追加 (現状 6 言語標準)
- オンサイト研修・現地導入支援
- 24 時間電話サポート (Premium SLA オプション)
- ハードウェア (タブレット / 端末) の調達・キッティング
- 機械学習モデルの独自カスタマイズ

## 8. 検収条件

以下のすべてを満たした時点で検収完了とする:

1. M3 本番稼働後、4 週間 (M4) の安定稼働
2. 重大障害 (Severity 1) の発生なし、または発生時に SLA 内で復旧
3. 全成果物 (§4) の納品完了
4. 受入テスト (UAT) 全項目の合格

委託者は M4 期間終了から 5 営業日以内に検収結果を書面通知する。期限内に通知がない場合、自動的に検収完了とみなす。

### 8.1 障害レベル定義
| Severity | 定義 | 対応開始 | 復旧目標 |
|---|---|---|---|
| S1 (重大) | サービス全体停止・データ消失 | 1 時間以内 | 4 時間以内 |
| S2 (高) | 主要機能停止・代替手段なし | 4 時間以内 | 1 営業日以内 |
| S3 (中) | 一部機能不具合・代替手段あり | 1 営業日以内 | 5 営業日以内 |
| S4 (低) | 軽微 / 表示不具合 | 5 営業日以内 | 次回リリース時 |
