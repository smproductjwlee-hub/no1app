    追加の管理者（PC）

    同じワークスペース名でログイン画面から「管理者」として入ると、同じ現場に別の管理者セッションが追加されます。パスワードは顧客向け共有パスワードです。

    用語・単語の登録（PC）

    Google スプレッドシート側でシートを編集してください。下の「シナリオ・用語」から API 経由の参照を確認できます。

    参加用 QR

    スマホで読み取る場合: .env の PUBLIC_BASE_URL を PC の LAN アドレス（例: http://192.168.0.5:8000）にし、サーバーを --host 0.0.0.0 で起動してください。127.0.0.1 はスマホ自身を指します。

    用語・単語登録

    Google スプレッドシート連携のデータを API で参照しています。編集はスプレッドシート側で行い、サーバー設定のスプレッドシートIDを参照してください。

    参照API: /api/v1/curriculum/food-glossary など
