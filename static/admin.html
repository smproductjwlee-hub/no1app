      スプレッドシートのタブ（横スクロール）

      追加の管理者（PC）

      同じワークスペース名でログイン画面から「管理者」として入ると、同じ現場に別の管理者セッションが追加されます。パスワードは顧客向け共有パスワードです。

      Google の編集に加え、「用語」「表現」追加から店舗専用データをSQLiteに保存できます（シート本体は変更しません）。「シナリオ・用語」で API を確認できます。
